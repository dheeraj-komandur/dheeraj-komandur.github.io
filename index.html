Research Paper

Introduced a novel system which uses deep learning algorithms and is integrated with smartphones to detect potholes in real-time. Deep learning object detection algorithm: Single Shot Multi-box Detector (SSD) looks for potholes using a mobile camera in the background. Accelerometer and gyroscope readings are continuously taken and assessed by a Artificial Neural Network model to detect registered potholes. This dual mechanism of camera based as well as accelerometer-gyroscope based detection cross validates detections and is zero cost solution for real-time pothole detection.
Won Best Paper Award at IEEE International Conference, IEEE PuneCon 2019

Indoor Navigation for University Campus

Built a indoor navigation web-app that calculates the shortest path from source to destination. A divide and conquer approach is implemented, a dijstra algorithm first solves the holistic maze (University Premises) and then sends input to each individual webpage(Each floor). These individual pages(each floor) solve their individual maze(entry and exit of that perticular floor). Dijstra algorithm is used to solve both holistic and local maze.
A user can scan barcode placed across campus and select it as the source location.
